🚀 如何使用
1. 克隆项目

git clone https://github.com/Zhangxulang/Python_Base.git
cd Python_Base
2. 运行示例代码

python day01/hello_world.py
3. 运行自动提交脚本（Windows）

tools/auto_push.bat
💡 学习建议
每天坚持完成一个子目录内容；

多写、多练、多出错、多总结；

推荐手写笔记或思维导图辅助记忆；

建议搭配 PyCharm + Terminal 环境练习。

📦 环境依赖
核心使用 Python 标准库，部分高级部分需要：

matplotlib

pandas

安装方式：

pip install matplotlib pandas
建议使用 Python 3.8+ 环境。
